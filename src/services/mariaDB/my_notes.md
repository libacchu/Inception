

# Types of Databases

1. Key-value
		eg. Redis
		stores database on RAM and is very fast
		used for cache

2. Wide Column
		eg. Hbase

3. Document
		eg. MongoDB, firestore

4. Relational Database
		eg. mySQL

5. Graphs
		eg. Neo4J

6. Search Engine
		eg. algolia, meilisearch

7. Multi-model
		eg. FaunaDB

8. Time-series
9. Data Warehouse
